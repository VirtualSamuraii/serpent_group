# Serpent C2

## Project setup

1/ Open the ```src/services/DashboardServices.js``` file and replace with your C2 Server IP Address

2/ Run : ```cd serpent_group/c2/web_app```

3/ Run : ```npm run install```

4/ Run : ```npm run build```

5/ Run : ```serve -S dist/```
